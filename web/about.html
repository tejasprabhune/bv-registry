what it is

bv is a tool manager for bioinformatics, written in Rust. It installs tools as OCI containers from a curated registry, pins them to content-addressed digests in bv.lock, and reproduces the pinned environment on any machine via bv sync. The same lockfile works on Docker and on Apptainer.

bv add <tool> writes bv.toml/bv.lock and generates PATH shims under .bv/bin/. bv exec <cmd> runs any command with those shims on PATH, so existing pipelines that call binaries by name (Make, Snakemake, Python subprocess) work without code changes.

install

curl -fsSL https://raw.githubusercontent.com/mlberkeley/bv/main/install.sh | sh
# or: cargo install biov

Runs on Linux and macOS. Windows: use WSL2 with Docker Desktop's WSL2 backend; native Windows is not yet supported.

tl;dr

bv add blast
bv exec blastn -query foo.fa -db nr
git add bv.toml bv.lock     # commit the pin; `bv sync` reproduces anywhere

container runtime

bv shells out to Docker or Apptainer. Install one of them first: 

# Docker, typical on a laptop
curl -fsSL https://get.docker.com/rootless | sh

# Apptainer, typical on HPC nodes (no daemon, no root)
conda install -c conda-forge apptainer

For GPU tools on Docker, install nvidia-container-toolkit. Apptainer uses the host's NVIDIA libraries with --nv.
